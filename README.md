# CachyOS-Real-Blue-Icons

A set of CachyOS icons that are actually blue.

The default CachyOS icons are commonly described as blue, but are in fact green. I modified these in inkscape to fit the blue aesthetic I was going for. The icons in this repository are true blue replacements for the default icons.

These icons can be used as overrides by placing them in the destination folder specified below.
